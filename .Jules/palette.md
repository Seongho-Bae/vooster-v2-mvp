## 2024-06-01 - UI-less Agent CLI Application
**Learning:** This repository (`vspec`) is a local-first, file-based CLI application specifically designed for AI agents. It does not have a web UI, HTML, CSS, or React components. Standard visual and web accessibility UX enhancements (like ARIA labels, hover states, etc.) are entirely inapplicable here.
**Action:** Do not attempt to implement standard web UX or visual changes on this codebase. Recognize the specific context of an agent-first CLI tool and stop execution without creating a PR when visual/accessibility enhancements are requested.
